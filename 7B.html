Marco teórico — Unit 7B

La unidad 7B trabaja principalmente el second conditional y la diferencia entre el first conditional y el second conditional. Además, el vocabulario de esta unidad se enfoca en el tema de houses.

1. Second conditional

El second conditional se usa para hablar de una situación hipotética, imaginaria o poco probable en el presente o en el futuro, y de su consecuencia.

Estructura principal: if + past simple, would / wouldn’t + verbo base

Ejemplos:

  • If I had a job, I’d get my own flat.
  • If David spoke good English, he could get a job in that hotel.
  • I wouldn’t do that job unless they paid me a really good salary.
Idea clave: el second conditional no habla de una situación real o segura. Habla de algo imaginado.

2. Uso de would / wouldn’t / could

En la segunda parte de la oración podemos usar would, wouldn’t o también could, dependiendo del significado.

  • If I had more time, I’d study more.
  • If it was warmer, we could have a swim.

3. Was / were

Después de if, con I, he, she, se puede usar was o were. Pero en inglés más formal, were suele sonar mejor.

  • If I were you, I’d buy a new computer.
  • If she were here, she’d know what to do.

4. If I were you...

Esta estructura se usa mucho para dar consejos.

If I were you, I’d + verbo

Ejemplos:

  • If I were you, I’d talk to the teacher.
  • If I were you, I wouldn’t buy that house.

5. Choosing between conditionals

First conditional

Se usa para una situación posible y real en el futuro.

Ejemplo: If I study, I’ll pass.

Second conditional

Se usa para una situación imaginaria o poco probable.

Ejemplo: If I had more time, I’d study French.

Resumen fácil:
First conditional = algo sí puede pasar.
Second conditional = algo imaginario o menos real.

Explicación sencilla

La 7B enseña a hablar de situaciones imaginarias. Si dices algo como “If I had a lot of money, I’d buy a big house”, no significa que tienes ese dinero. Solo lo estás imaginando.

Ejemplos fáciles:

  • If I had a car, I’d drive to school. → Si tuviera carro, iría al colegio en carro.
  • If I lived in the country, I’d be happier. → Si viviera en el campo, sería más feliz.
  • If I were you, I’d study more. → Si yo fuera tú, estudiaría más.
Truco rápido:
  • if + past simple
  • would + verbo
  • If I were you... = consejo

Estructuras principales

Estructura Uso Ejemplo
if + past simple, would + verbo Situación imaginaria If I had a job, I’d get my own flat.
if + past simple, wouldn’t + verbo Consecuencia negativa imaginaria I wouldn’t buy that bike if I were you.
if + past simple, could + verbo Posibilidad o capacidad imaginaria If it was warmer, we could swim.
If I were you, I’d + verbo Dar consejo If I were you, I’d sell the house.

Errores comunes

  • ❌ If I would have time, I would go.
  • ✅ If I had time, I would go.
  • ❌ If I were you, I will study.
  • ✅ If I were you, I’d study.
  • ❌ If she has a house, she’d be happy.
  • ✅ If she had a house, she’d be happy.
